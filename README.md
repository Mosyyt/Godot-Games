## What is this?

Five 2D games made in Godot Engine as learning projects. Each game lives in its own folder and covers a different genre — fighting, space shooting, platforming, endless running, and puzzle. Two additional larger games are hosted externally on MEGA.

---

### fight

A 2D fighting game.

```bash
# Open fight/ in Godot, then press F5 to run
```

## Setup

1. Download and install [Godot Engine](https://godotengine.org/download)
2. Open Godot and import the project folder
3. Press F5 to run
